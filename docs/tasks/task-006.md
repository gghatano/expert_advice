# Task 006: 季節性プロファイルExpert実装

## 概要
季節性分解ベースのExpertを実装する。（spec §7.2-D）

## 成果物
- `src/experts/seasonal_profile.py`:
  - `STLSeasonalMean`: 曜日×時刻(0-23)の平均プロファイルをtrain期間から算出
    - 予測時に該当する曜日×時刻のプロファイル値を返す
    - オプション: 直近の残差トレンドで補正

## 受け入れ条件
- BaseExpertインターフェース準拠
- fit時にtrain期間から曜日×時刻プロファイルを学習（7×24=168パターン）
- predict_nextは対応するプロファイル値を返す
- テストが通る
